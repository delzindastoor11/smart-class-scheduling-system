# Smart Class Scheduler

Smart Class Scheduler is a **Java-based web application** designed to manage academic scheduling efficiently.  
It provides an admin interface to handle teachers, subjects, classrooms, timetables, and notices in a centralized system.

 ## Key Features
- Admin dashboard for managing academic resources
- Teacher, subject, classroom, and timetable management
- Notice management system
- Role-based access control
- Clean and modular architecture

 ## Tech Stack
- **Java (JSP & Servlets)**
- **MVC Architecture**
- **DAO Pattern**
- **Apache Tomcat Server**
- **MySQL (MySQL Workbench)**
- **HTML, CSS, JavaScript**
- **Maven**

## Architecture
- **MVC (Model–View–Controller)** for separation of concerns  
- **DAO (Data Access Object)** for database operations  
- **Servlets** as controllers  
- **JSP** for the view layer  

## Database
- MySQL database schema and sample data are included in:
- database/smart_class_scheduler.sql
 Database designed using **MySQL Workbench**

## How to Run
1. Import the project into NetBeans
2. Import the MySQL database using the provided `.sql` file
3. Configure database credentials
4. Deploy the project on **Apache Tomcat**
5. Access the application via `localhost:8080`
